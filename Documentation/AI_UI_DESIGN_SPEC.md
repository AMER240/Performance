## ?? User Experience Flow

1. **User clicks "Get Suggestions"**
   - Button disabled
   - Text changes to "Analyzing..."
   - Progress bar appears (marquee style)
   
2. **API call in progress**
   - Progress bar animating
   - Results container shows "Loading..."
   
3. **Results received**
   - Progress bar hidden
   - Results fade in one by one
   - Each category color-coded
   - Scroll to top automatically
   
4. **Error state**
   - Progress bar hidden
   - Error message in red
   - Retry button appears
